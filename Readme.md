# Setup

```
$ python3 -m venv .venv
$ . .venv/bin/activate
$ pip install -r requirements.txt
```

# Run

```
$ python app.py
```
# Unit Tests

```
$ python test_products_db.py
```

# Integration Tests

(while app is running - do this in another tab/window - this will change data, of course)

```
$ python test_app.py
```
